translate¶

dragonfly-doe2 translate¶

Commands for translating Dragonfly JSON files to *.inp files.

dragonfly-doe2 translate [OPTIONS] COMMAND [ARGS]...

dfjson-to-inp¶

Translate a DFJSON into a doe2 *.inp file.

dragonfly-doe2 translate dfjson-to-inp [OPTIONS] DF_JSON

Options

-n, --name <name>¶

Name of the output file.

Default:

model

-f, --folder <folder>¶

Path to target folder.

Default:

.

Arguments

DF_JSON¶

Required argument

hbjson-to-inp¶

Translate a HBJSON into a doe2 *.inp file.

dragonfly-doe2 translate hbjson-to-inp [OPTIONS] HB_JSON

Options

-n, --name <name>¶

Name of the output file.

Default:

model

-f, --folder <folder>¶

Path to target folder.

Default:

.

Arguments

Required argument
